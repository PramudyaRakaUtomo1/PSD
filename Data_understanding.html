Data Understanding

  • Dataset berisi 150 sampel dengan 4 fitur numerik (sepal_length, sepal_width, petal_length, petal_width) dan 1 label Class.
  • Tidak ditemukan missing value atau duplikasi; tiap kelas terdistribusi seimbang (50 sampel per kelas).
  • Skema / kolom (dari file kamu): id (int) — identifier baris Class (object/string) — label klasifikasi (Iris-setosa, Iris-versicolor, Iris-virginica) sepal_length (float) sepal_width (float) petal_length (float) petal_width (float)
  • Ukuran dataset: 150 baris × 6 kolom. Missing values / duplicates: tidak ada missing value (semua kolom terisi) dan tidak ada baris duplikat. Distribusi kelas: seimbang — tiap kelas 50 sampel (Iris-setosa 50, Iris-versicolor 50, Iris-virginica 50). Statistik ringkas (min / max / mean) — empat fitur numerik: sepal_length: min = 4.3, max = 7.9, mean ≈ 5.84 sepal_width: min = 2.0, max = 4.4, mean ≈ 3.05 petal_length: min = 1.0, max = 6.9, mean ≈ 3.76 petal_width: min = 0.1, max = 2.5, mean ≈ 1.20 Rata-rata tiap kelas (class-wise means): Iris-setosa: sepal_length ≈ 5.006, sepal_width ≈ 3.418, petal_length ≈ 1.464, petal_width ≈ 0.244 Iris-versicolor: sepal_length ≈ 5.936, sepal_width ≈ 2.770, petal_length ≈ 4.260, petal_width ≈ 1.326 Iris-virginica: sepal_length ≈ 6.588, sepal_width ≈ 2.974, petal_length ≈ 5.552, petal_width ≈ 2.026
