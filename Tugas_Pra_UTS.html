Tugas Pra UTS

  • Import Dataset Ecoli

    • Import dataset ke dalam postgre

  • Membuat Project Pra UTS di Knime

    • disini diberi nama Project Pra UTS

  • Menyambungkan postgre dengan Knime

    • pilih node 'postgreSQL Connector'

    Kemudian isi hostname dan nama database yang ingin digunakan

  • Memilih table yang ingin digunakan

    • pilih node 'DB Table Selector' dan pilih table mana yang digunakan

  • Menampilkan tabel yang telah digunakan

    • untuk menampilkan tabel bisa menggunakan node 'DB Reader'

  • Unbalancing dataset

    • menggunakan node 'Value Counter'

  • Mencari missing values

    • pilih node 'Missing values'

  • Deteksi outlier

    • Untuk mendeteksi outlier disini menggunakan node 'Numeric outlier'
